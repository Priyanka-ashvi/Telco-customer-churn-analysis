# Telco_Customer_Churn_Analysis
Analyzed telecom customer data to identify key factors driving churn using Python. Performed Exploratory Data Analysis (EDA) with Pandas, Matplotlib, and Seaborn to uncover insights on customer behavior, payment methods, and contract types.
Project Overview

This project focuses on analyzing customer churn in a telecom company to identify the key factors influencing customer retention and attrition. Using Python-based Exploratory Data Analysis (EDA) techniques, we explore how customer demographics, services, billing methods, and contract types impact churn behavior.

The analysis provides data-driven insights to help telecom providers improve retention, reduce customer loss, and enhance business profitability.
